
# Regional Scrum Gathering Lisbon 2026 🇵🇹

Landing page moderna e responsiva para o evento **RSG Lisbon 2026**, organizada pela **TugÁgil**.

## 🚀 Tecnologias
- **React 19**
- **Tailwind CSS** (Estilização)
- **Lucide React** (Ícones)
- **Vite** (Build Tool)
- **Vercel Serverless Functions** (Envio de E-mail)
- **Resend** (Infraestrutura de E-mail)

## 🛠️ Configuração de Produção (Vercel)
Para que os formulários funcionem corretamente, adicione a seguinte variável de ambiente no seu projeto na Vercel:
1. Vá a **Settings > Environment Variables**.
2. Adicione `RESEND_API_KEY` com o valor da sua chave obtida em [resend.com](https://resend.com).

## 👨‍💻 Admin Area (Secret Access)
Pode aceder à área de gestão de leads de duas formas:
1. **Atalho de Teclado:** Pressione as teclas `A` + `D` + `M` simultaneamente.
2. **Botão Discreto:** Existe um ícone de engrenagem quase invisível no canto inferior esquerdo do rodapé.

## 📦 Como rodar localmente
```bash
npm install
npm run dev
```
